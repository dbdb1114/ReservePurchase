## 예약구매 서비스

### 개요
예약구매 서비스란 E-COMMERCE 플랫폼 중에서 특정 시간대에 열리는 티켓 판매 서비스입니다.
1. 특정 시간에 몇 가지 상품군을 오픈합니다.
2. 소비자가 구매를 시도하여, 결제 시도 프로세스에 진입합니다.
   - 구매를 시도하고있는 상품수 만큼 재고를 감소시킵니다.
   - 고객 이탈 발생시 즉시 재고를 복구시킵니다.
